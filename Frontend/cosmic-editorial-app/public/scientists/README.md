## 🚀 Cara Download

### Option A: Browser (Manual)
1. Buka tiap link di browser
2. Right-click → Save image as...
3. Rename sesuai nama file di atas
4. Simpan di folder ini

### Option B: Terminal (Mac/Linux)
```bash
cd public/scientists

# Klasik
curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/3/3b/Portrait_of_Sir_Isaac_Newton%2C_1689.jpg/440px-Portrait_of_Sir_Isaac_Newton%2C_1689.jpg" && mv 440px-Portrait_of_Sir_Isaac_Newton%2C_1689.jpg newton.jpg

curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/d/d4/Johannes_Kepler_1610.jpg/440px-Johannes_Kepler_1610.jpg" && mv 440px-Johannes_Kepler_1610.jpg kepler.jpg

curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/d/d4/Galileo_Galilei_2.jpg/440px-Galileo_Galilei_2.jpg" && mv 440px-Galileo_Galilei_2.jpg galileo.jpg

# Modern
curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Einstein_1921_by_F_Schmutzer_-_restoration.jpg/440px-Einstein_1921_by_F_Schmutzer_-_restoration.jpg" && mv 440px-Einstein_1921_by_F_Schmutzer_-_restoration.jpg einstein.jpg

curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Max_Planck_1933.jpg/440px-Max_Planck_1933.jpg" && mv 440px-Max_Planck_1933.jpg planck.jpg

curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/6/6d/Niels_Bohr.jpg/440px-Niels_Bohr.jpg" && mv 440px-Niels_Bohr.jpg bohr.jpg

# Astronomi
curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/1/14/Edwin_Hubble.jpg/440px-Edwin_Hubble.jpg" && mv 440px-Edwin_Hubble.jpg hubble.jpg

curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/e/eb/Stephen_Hawking.StarChild.jpg/440px-Stephen_Hawking.StarChild.jpg" && mv 440px-Stephen_Hawking.StarChild.jpg hawking.jpg

curl -O "https://upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Carl_Sagan_in_1987.jpg/440px-Carl_Sagan_in_1987.jpg" && mv 440px-Carl_Sagan_in_1987.jpg sagan.jpg
```
